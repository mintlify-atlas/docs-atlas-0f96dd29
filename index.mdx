    <Step title="Install the extension">
      Clone the repository and load it as an unpacked Chrome extension
      
      ```bash
      git clone https://github.com/apurv-1/mintlify-oneclick-extension.git
      ```
      
      <Note>
        Open Chrome, navigate to `chrome://extensions`, enable Developer mode, and click "Load unpacked" to select the cloned folder.
      </Note>
    </Step>
    
    <Step title="Visit any GitHub repository">
      Navigate to any GitHub repository page in your browser
      
      The extension automatically detects valid repository pages and injects the "View Docs on Mintlify" button next to the Watch, Fork, and Star buttons.
    </Step>
    
    <Step title="Access instant documentation">
      Click the button to view auto-generated Mintlify documentation
      
      ```
      github.com/facebook/react → mintlify.com/facebook/react
      ```
      
      <Tip>
        The extension works with all public GitHub repositories and intelligently filters out non-repository pages like settings, marketplace, and trending.
      </Tip>
    </Step>
  </Steps>
