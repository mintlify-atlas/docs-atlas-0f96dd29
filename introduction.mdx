The Mintlify One-Click Extension adds a "View Docs on Mintlify" button to every GitHub repository page. When you click it, the extension instantly opens auto-generated, production-quality documentation for that repository on Mintlify.

## What problem does it solve?

Navigating unfamiliar codebases on GitHub can be time-consuming and overwhelming. Reading through source files to understand structure, APIs, and functionality requires significant effort. This extension bridges that gap by providing instant access to comprehensive, AI-generated documentation for any public GitHub repository.

## How it works

The extension operates through a simple workflow:

<Steps>
  <Step title="Detect GitHub repositories">
    The extension monitors when you visit GitHub repository pages and automatically identifies the owner and repository name from the URL.
  </Step>
  
  <Step title="Add action button">
    A "View Docs on Mintlify" button is injected into the repository's action bar, alongside the Watch, Fork, and Star buttons.
  </Step>
  
  <Step title="Generate documentation URL">
    When clicked, the extension transforms the GitHub URL into a Mintlify documentation URL using the pattern: `github.com/owner/repo` → `mintlify.com/owner/repo`
  </Step>
  
  <Step title="Open documentation">
    Your browser opens the Mintlify documentation page in a new tab, where you can explore the auto-generated docs.
  </Step>
</Steps>
