## Filtered Pages

The extension intelligently filters out GitHub pages that aren't repositories. The button **will not appear** on these pages:

<AccordionGroup>
  <Accordion title="Settings Pages">
    ```text
    https://github.com/settings/profile
    ```
    The extension recognizes "settings" as a reserved path and won't inject the button.
  </Accordion>

  <Accordion title="GitHub Marketplace">
    ```text
    https://github.com/marketplace
    ```
    Marketplace pages are filtered out.
  </Accordion>

  <Accordion title="Explore and Trending">
    ```text
    https://github.com/explore
    https://github.com/trending
    ```
    These discovery pages don't represent specific repositories.
  </Accordion>

  <Accordion title="Authentication Pages">
    ```text
    https://github.com/login
    https://github.com/signup
    ```
    Login and signup pages are excluded.
  </Accordion>
</AccordionGroup>

### Complete List of Reserved Paths

The extension filters out URLs starting with these paths:

```javascript
const reserved = [
  "settings", "marketplace", "explore", "topics", "trending",
  "collections", "events", "sponsors", "login", "signup",
  "notifications", "new", "organizations", "features", "pricing",
];
```

## Edge Cases

### Invalid Repository URLs

The button won't appear if the URL doesn't contain both an owner and repository name:

<CodeGroup>
```text Invalid - No Repository
https://github.com/facebook
```

```text Valid - Owner and Repo
https://github.com/facebook/react
```
</CodeGroup>

### Trailing Slashes

The extension handles trailing slashes automatically:

<CodeGroup>
```text Without Trailing Slash
https://github.com/vercel/next.js
```

```text With Trailing Slash
https://github.com/vercel/next.js/
```
</CodeGroup>

Both URLs produce the same Mintlify link: `https://www.mintlify.com/vercel/next.js`

## Using the Popup for Direct Access

The extension popup shows the current repository and provides a direct link:

<Steps>
  <Step title="Navigate to any repository page">
    For example: `https://github.com/facebook/react/blob/main/README.md`
  </Step>
  
  <Step title="Click the extension icon">
    The popup opens showing "facebook/react" as the repository name.
  </Step>
  
  <Step title="Click the popup link">
    The popup's "View Docs on Mintlify" button opens the docs in a new tab.
  </Step>
</Steps>

<Tip>
The popup provides an alternative way to access Mintlify docs even when the extension is disabled, as long as you're on a valid GitHub repository page.
</Tip>
